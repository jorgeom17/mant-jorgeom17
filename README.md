# mant-jorgeom17
Este proyecto consiste en un repositorio donde se creó una librería para generar contraseñas aleatoriamente

### Pre requisitos:

El proyecto se puede descargar desde [github](https://github.com/jorgeom77/mant-jorgeom17) o desde la aplicación de GitHub Desktop.
Para ejecutar el programa, se necesita descargar la versión 3.7.2 de python desde [Python](https://www.python.org/ ); no obstante, también funciona con versiones anteriores.

### Ejecución del proyecto:
Para utilizar la función, se debe llamar al método generatePassword(), el cuál retorna una contraseña con las siguientes características:
	- Longitud entre 4 y 16 caracteres.
	- Puede incluir mayúsculas, minúsculas, números y caracteres especiales.
